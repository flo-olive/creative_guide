# FLO Creative System — Claude Skill

FLO 서비스 UI 개발을 위한 Claude 디자인 시스템 스킬입니다.
바이브코딩 시 이 스킬을 통해 FLO Figma 디자인 스펙을 자동으로 따릅니다.

## 포함 내용

- 컬러 시스템 (Blue / Teal / Pink / Yellow 4가지 테마)
- 타이포그래피 (Pretendard 폰트 규칙)
- 컴포넌트 스펙 (카드, 태그, 입력창, CTA 버튼, 플레이리스트)
- 라이트모드 강제 적용 규칙
- 인터랙션 / Shadow 임의 생성 금지 규칙
- 아이콘 & 로딩 애니메이션 에셋 가이드 (flo-olive/creative_guide)

## 설치 방법

터미널에서 아래 명령어를 실행하세요:

```bash
claude skill install https://github.com/YOUR_USERNAME/flo-creative-system
```

또는 직접 클론해서 Claude skills 폴더에 추가:

```bash
git clone https://github.com/YOUR_USERNAME/flo-creative-system ~/.claude/skills/flo-creative-system
```

## 사용 방법

설치 후 Claude와 대화할 때 자동으로 적용됩니다.
예: "플레이리스트 컴포넌트 만들어줘", "카드 UI 구현해줘" 등
